# ERP Chat App with WhatsApp Integration

## Features
- Real-time chat using WebSocket
- Private messaging
- MySQL database storage
- WhatsApp integration (Twilio)

## Tech Stack
- Java (Servlet + JSP)
- WebSocket
- MySQL
- Twilio API

## How to Run
1. Import into Eclipse
2. Configure Tomcat
3. Setup database
4. Run project

## Future Scope
- File sharing
- Online users
- Chat UI improvement
